Detailed Description

template<typename T>
class APPRSDK::FunctionSystemBase< T >

FunctionSystemBase is an abstract class for function system classes.

The FunctionSystemBase abstract class has to be implemented by all concrete FunctionSystem classes. All the methods are inhereted from IFunctionSystem, and these are complemented with protected data members. Where possible, common method definitions are given in the accompanying FunctionSystemBase.cpp file. FunctionSystemBase retains the following template parameter T : type of the signal and function system used for the approximation.
