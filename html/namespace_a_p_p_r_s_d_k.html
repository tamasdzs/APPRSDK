Detailed Description

ApproxStat class.

The ApproxStat class implements functions that return different. Statistical information about the state of an approximation. These include:

  • PRD
  • Estimate of standard deviation
  • The number of iterations for the optimazation
  • Any exit message provided by the optimazation algorithm
