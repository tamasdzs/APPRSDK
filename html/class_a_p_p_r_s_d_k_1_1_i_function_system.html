Detailed Description

template<typename T>
class APPRSDK::IFunctionSystem< T >

Interface for function system classes.

The IApproximator interface has to be implemented by all concrete FunctionSystem classes. The interface has a template parameter: T : type of the signal and function system used for the approximation.
