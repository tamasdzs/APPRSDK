Detailed Description

template<typename T>
class APPRSDK::VariableProjection< T >

VariableProjection class Implements the variable projection algorithm.

The VariableProjection class implements the IOptimazible interface. VariableProjection type objects are capable of conducting an approximation with different parameters, and sharing statistical information about it.
